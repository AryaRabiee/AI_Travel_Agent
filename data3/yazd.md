# یزد

## Profile
- Best season: پاییز و بهار
- Weather: گرم و خشک، نیمه‌بیابانی
- Suitable days: ۲ تا ۴ روز
- Budget level: متوسط
- Interests: تاریخی، معماری سنتی، بادگیرها، کویرگردی
- Travel style: گردش تاریخی، بازدید فرهنگی، پیاده‌گردی شهری

## Description
یزد یکی از قدیمی‌ترین شهرهای ایران است که با بافت سنتی و معماری خشتی معروف است.
این شهر دارای بادگیرهای متعدد، خانه‌های تاریخی، مسجدها و باغ‌های قدیمی است.
آب و هوای یزد گرم و خشک بوده و تابستان‌ها بسیار گرم و زمستان‌ها معتدل است.
یزد به عنوان شهر بادگیرها و مرکز فرهنگ کویری ایران شناخته می‌شود.
